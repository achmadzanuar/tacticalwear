URCON - Tactical Wear (Vite + React)

Quick start:

1. cd urcon
2. npm install
3. npm run dev

Notes:
- Contentful credentials are stored in `.env`.
- Product data is fetched from Contentful content type `tacticalwear` and uses fields: name, description, price, category, image, featured.
- CTA buttons open WhatsApp (prefill) to +62813330861324.
